Epeak is an AI coaching platform for competitive esports players. It analyzes match performance, identifies weaknesses, and delivers personalized improvement plans in English and Arabic. As players grow, the platform surfaces rising talent for esports scouts and academies — creating the first end-to-end player development ecosystem built for Saudi Arabia. The prototype should be able to analyze a match and provide a personalized improvement plan in English and Arabic. It should also be able to surface rising talent for esports scouts and academies. Starting with the game Valorant.
